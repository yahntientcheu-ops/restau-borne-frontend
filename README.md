# Frontend - Borne de Commande (Groupe BMW)

## Description
Interface tactile pour borne de commande réalisée en JavaFX.

## Prérequis
* Java 21
* Maven

## Installation et Lancement
1. **Lancement** : `mvn javafx:run`
2. **Configuration** : Assurez-vous que le Backend est lancé pour que l'interface puisse récupérer les données.

## Fonctionnalités incluses
* Sélection de catégories puis les plats.
* Personnalisation (quantité, options).
* Récapitulatif et confirmation avec numéro de commande.
